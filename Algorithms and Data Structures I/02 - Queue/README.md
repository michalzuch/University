### Interface

```c++
class Queue {
    // Default constructor
    Queue();

    // Inserts x element into the queue (also enqueue)
    void push(int x);
    
    // Removes an item from the queue and returns its value (dequeue)
    int pop();
    
    // Returns the number of items in a queue
    int size();
    
    // Checks if the queue is empty
    bool empty();
};
```
